# Inspiration

This folder contains reference materials and examples that inspired this project.

## LlamaIndex Notebook

**File**: [Basic_RAG_With_LlamaIndex.ipynb](Basic_RAG_With_LlamaIndex.ipynb)

**Source**: [Anthropic Claude Cookbooks - LlamaIndex](https://github.com/anthropics/claude-cookbooks/tree/main/third_party/LlamaIndex)

**Description**: Official Anthropic cookbook demonstrating basic RAG implementation with LlamaIndex and Claude. This notebook provided the foundation for understanding:
- LlamaIndex setup and configuration
- Document loading and indexing patterns
- Query engine usage
- Integration with Claude via Anthropic API

**How It Influenced This Project**:
- Inspired the single-document RAG approach in [simple_rag.py](../simple_rag.py)
- Demonstrated LlamaIndex's clean API for RAG pipelines
- Showed best practices for Claude + LlamaIndex integration
- Provided baseline for chunk size and retrieval parameter tuning
